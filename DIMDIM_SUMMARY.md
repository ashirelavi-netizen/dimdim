## דיאלוג הגדרות

### ריבוע 1 — הגדרות כלליות
- שכבה (נבחרת מראש לפי ערך שמור)
- סגנון מידה (נבחר מראש לפי ערך שמור)
- קנה מידה
- מרחק קו המידה (× גובה טקסט)

### ריבוע 2 — שכבות חוצות / קרובות
כפתור "+ הוסף שכבה" — פותח דיאלוג בחירת שכבה עם:
- שדה סינון — מתחיל לסנן מ-3 תווים
- סינון לפי מספר מילות חיפוש מופרדות ברווח (
OR
)
- רשימת שכבות

---

## DIMDIMUNGROUP

1. בחירת מידה
2. קריאת מזהה קבוצה מה-
`xdata`
3. חיפוש כל המידות מאותה קבוצה (
`ssget "X"` + סינון)
4. שאלה: `[Delete/Ungroup]`
   - `Delete` — מוחק את כל הקבוצה
   - `Ungroup` — מנתק (מסיר
`xdata`
)

## פונקציות עזר מרכזיות

| פונקציה | תפקיד |
|---------|--------|
| `ddim:get-settings` / `ddim:put-settings` | קריאה/כתיבה להגדרות |
| `ddim:default-settings` | ערכי ברירת מחדל |
| `ddim:layer-list` / `ddim:style-list` | רשימת שכבות/סגנונות |
| `ddim:split-words` | פיצול מחרוזת לפי רווחים |
| `ddim:word-match` | בדיקה אם פריט מכיל אחת מהמילים |
| `ddim:filter-by-words` | סינון רשימה לפי מילות חיפוש (OR) — שיטה כללית |
| `ddim:str-to-list` / `ddim:list-to-str` | פיצול/איחוד רשימת שכבות לפי ; |
| `ddim:pick-layer` | דיאלוג בחירת שכבה |
| `ddim:dlg` | דיאלוג הגדרות ראשי |
| `ddim:find-cross-pts` | נקודות חיתוך עם XLINE |
| `ddim:find-near-pts` | נקודות קרובות ל-XLINE |
| `ddim:filter-in-range` | סינון נקודות לתחום שני הקליקים |
| `ddim:sort-dedup` | מיון והסרת כפילויות |
| `ddim:pick-xline` | מיקום XLINE עם תצוגה חיה |
| `ddim:create-dims` | יצירת מידות + תיוג xdata |
| `ddim:tag` / `ddim:newid` | תיוג קבוצה ב-xdata |
| `ddim:get-dimtxt` | קריאת גובה טקסט מסגנון מידה |
