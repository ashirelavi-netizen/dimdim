## זרימת עבודה

### תפריט ראשי
`QDims | Settings | Ungroup/Delete`

### זרימת QDims

**שלב 1 — מיקום XLINE**
1. קליק ראשון (עם OSNAP) — נקודת ייחוס
2. XLINE עוקב אחרי העכבר בזמן אמת (`grread t`), מחליט H/V לפי כיוון תנועה
3. קליק שני (עם OSNAP) — קובע מיקום
4. שאלת אישור — `No` חוזר לשלב 3 בלבד
5. XLINE נמחק אחרי אישור

**שלב 2 — מציאת נקודות**
- שכבות חוצות: נקודות חיתוך על ה-XLINE — **רק מסגמנטים ניצבים בדיוק** לכיוון קו המידה
- שכבות קרובות: נקודות קצה אמיתיות של אובייקטים בטווח (לא הטלה)
- סינון לתחום בין שני הקליקים
- מיון + הסרת כפילויות (סבלנות 0.1 יחידות)

**שלב 3 — יצירת מידות**
- DIMLINEAR אוטומטי על כל זוג נקודות
- כל המידות נכנסות לגרופ אחד עם שם `DDIM-<millisecs>-<counter>`

### זרימת Ungroup/Delete
1. לחיצה על `Ungroup/Delete` בתפריט
2. בחירת קו מידה בציור
3. פופ-אפ קטן ליד הסמן:
   - **Ungroup** — מוחק את אובייקט הגרופ, קווי המידה נשארים שלמים
   - **Delete** — מוחק את הגרופ ואת כל קווי המידה שבו
   - **Cancel** — ביטול

## פונקציות — תיאור מדויק

### הגדרות
| פונקציה | תיאור |
|---------|-------|
| `ddim:get-settings` | קוראת הגדרות מה-Named Object Dictionary. מחזירה רשימה של 8 מחרוזות, או nil אם לא קיים |
| `ddim:put-settings lst` | כותבת רשימת הגדרות ל-Dictionary. מוחקת ערך קיים לפני כתיבה |
| `ddim:default-settings` | מחזירה ערכי ברירת מחדל: `("0" "100.0" "0" "50" "" "" "3.0" "7")` |

### רשימות
| פונקציה | תיאור |
|---------|-------|
| `ddim:layer-list` | מחזירה רשימת שמות שכבות ממויינת |
| `ddim:style-list` | מחזירה רשימת שמות סגנונות מידה ממויינת |

### גרופ
| פונקציה | תיאור |
|---------|-------|
| `ddim:make-group ss grp-name` | יוצרת גרופ VLA מ-selection set. משתמשת ב-`vla-add` + `vla-appenditems`. מדפיסה שגיאה אם נכשל |
| `ddim:find-group ent` | מחפשת בכל הגרופים עם קידומת `DDIM-` את הגרופ שמכיל את הישות. השוואה לפי handle (לא ename). מחזירה VLA Group object או nil |

### פופ-אפ
| פונקציה | תיאור |
|---------|-------|
| `ddim:write-popup-dcl` | כותבת DCL זמני לחלונית Ungroup/Delete/Cancel |
| `ddim:show-popup cpos` | מציגה חלונית קטנה ליד מיקום הסמן. מחזירה 1=Ungroup, 2=Delete, 0=Cancel |

### מחרוזות ושכבות
| פונקציה | תיאור |
|---------|-------|
| `ddim:list-to-str lst` | מחבר רשימה למחרוזת `"א;ב;ג;"` |
| `ddim:str-to-list str` | מפצל מחרוזת `"א;ב;ג;"` לרשימה |
| `ddim:split-words str` | מפצל מחרוזת לפי רווחים |
| `ddim:word-match item toks` | בודק אם item מכיל לפחות מילה אחת מ-toks (wcmatch) |
| `ddim:filter-by-words lst search` | מסנן רשימה לפי מילות חיפוש (OR) |

### דיאלוגים
| פונקציה | תיאור |
|---------|-------|
| `ddim:pick-layer path lays` | דיאלוג בחירת שכבה עם שדה סינון (מינימום 3 תווים) |
| `ddim:write-dcl` | כותבת DCL זמני לדיאלוג ההגדרות הראשי |
| `ddim:dlg-screen-xy` | מחשבת מיקום מסך לדיאלוג הגדרות (60%/55% מחלון התוכנה) |
| `ddim:dlg cur` | דיאלוג הגדרות ראשי. מקבל הגדרות נוכחיות, מחזיר רשימה חדשה או nil אם בוטל |

### גיאומטריה
| פונקציה | תיאור |
|---------|-------|
| `ddim:seg-is-perp p1 p2 dir` | **חדש.** בודק ניצבות בדיוק: H-xline ← `dx < 1e-6`; V-xline ← `dy < 1e-6` |
| `ddim:seg-xline-isect p1 p2 dir pos` | מחשב נקודת חיתוך של סגמנט עם XLINE. מחזיר נקודת 3D או nil |
| `ddim:dist-to-xline pt dir pos` | מרחק נקודה מה-XLINE |
| `ddim:closest-on-seg p1 p2 dir pos` | מחזיר את נקודת הקצה הקרובה יותר ל-XLINE |
| `ddim:entity-segs ed` | מחלץ רשימת סגמנטים מישות. תומך: LINE, LWPOLYLINE (כולל סגורה) |

### מציאת נקודות
| פונקציה | תיאור |
|---------|-------|
| `ddim:find-cross-pts dir pos layers` | מוצאת נקודות חיתוך על ה-XLINE מהשכבות החוצות — **רק סגמנטים ניצבים** |
| `ddim:find-near-pts dir pos layers dist` | מוצאת נקודות קצה אמיתיות של סגמנטים שלא חוצים את ה-XLINE אך נמצאים בטווח dist |
| `ddim:filter-in-range pts dir pt1 pt2` | מסננת נקודות לתחום בין שני הקליקים |
| `ddim:sort-dedup pts dir tol` | ממיינת נקודות ומסירה כפילויות בסבלנות tol |

### XLINE ומידות
| פונקציה | תיאור |
|---------|-------|
| `ddim:pick-xline layer xline-color` | ממקם XLINE עם תצוגה חיה. מטפל בביטול/ESC (מוחק XLINE). מחזיר `(dir pos pt1 pt2)` |
| `ddim:create-dims pts dir dim-style dim-layer dim-pt scale grp-id` | יוצרת DIMLINEAR על כל זוג נקודות, אוספת ל-ss, קוראת ל-ddim:make-group |
| `ddim:run-line s` | לוגיקה ראשית של יצירת מידות: קוראת לכל השלבים לפי סדר |
| `ddim:get-dimtxt style` | קורא גובה טקסט מסגנון מידה. ברירת מחדל 2.5 אם לא מוגדר |

## הערות טכניות חשובות

- גרופים מזוהים לפי קידומת `DDIM-` בשמם
- חיפוש גרופ לפי **handle** (לא ename) — אמין יותר ב-VLA
- `command "_.GROUP"` לא שמיש ב-ZCAD — פותח חלון דיאלוג
- `vla-delete` על Group מסיר את הגרופ בלבד — הישויות שבו נשארות שלמות
- `getvar "CURSORPOS"` משמש למיקום הפופ-אפ ליד הסמן
- קנה מידה: `actual-dist = base-dist × (scale / 50)`
